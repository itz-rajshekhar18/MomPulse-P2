## Data Flow

```
User logs cycle
      ↓
Saved to Firestore
      ↓
usePeriodPrediction() hook
      ↓
Fetches cycles from Firestore
      ↓
Calls ML Backend API
      ↓
ML Model predicts:
  - Next period dates
  - Ovulation date
  - Fertile window
  - Cycle regularity
  - Confidence level
      ↓
Predictions returned to components
      ↓
FertilityCard & Insights display real data
```

## Phase Calculation Logic

```typescript
Day 1-5:   Menstrual Phase
Day 6-13:  Follicular Phase
Day 14-16: Ovulation Phase
Day 17-28: Luteal Phase
```

## Confidence Levels

- **High:** 6+ cycles logged
- **Medium:** 3-5 cycles logged
- **Low:** 1-2 cycles logged

## Regularity Classification

- **Very Regular:** Std dev ≤ 2 days
- **Mostly Regular:** Std dev ≤ 5 days
- **Somewhat Irregular:** Std dev ≤ 8 days
- **Irregular:** Std dev > 8 days
