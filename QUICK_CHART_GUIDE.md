# 📊 Quick Chart Guide

## ✅ What Was Added
A beautiful cycle trend chart on the calendar page!

---

## 🚀 Quick Start

### 1. View the Chart
```
Navigate to: /dashboard/period/calendar
Scroll down below the calendar
```

### 2. See It in Action
```
Log 2+ cycles → Chart appears with trends
Log 6+ cycles → Full 6-month visualization
```

### 3. Interact
```
Hover → See detailed tooltips
Click [Day View] → See specific dates
Click [Month View] → See monthly overview
```

---

## 📊 What You'll See

### Statistics (Top)
- **Avg Cycle**: 28 days (typical)
- **Avg Period**: 5 days (typical)
- **Total Cycles**: Number logged
- **Regularity**: ✓ Regular or ~ Variable

### Chart (Middle)
- **Purple Line**: Cycle length (period to period)
- **Pink Line**: Period length (bleeding days)
- **Gradient Fill**: Visual emphasis
- **Grid Lines**: Reference markers

### Legend (Bottom)
- Explains what each line means
- Shows typical ranges
- Educational information

---

## 🎯 Key Features

1. **Interactive Tooltips** - Hover for details
2. **View Toggle** - Day or Month view
3. **Empty State** - Helpful when no data
4. **Responsive** - Works on all devices
5. **Animated** - Smooth transitions
6. **Educational** - Learn about your cycles

---

## 💡 What It Shows

### Cycle Length (Purple)
- Days from one period to the next
- Typical: 21-35 days
- Shows regularity

### Period Length (Pink)
- Days of bleeding
- Typical: 3-7 days
- Shows consistency
